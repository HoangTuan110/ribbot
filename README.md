## Installation

Make sure you have [Go](https://go.dev/dl/) and [`gum`](https://github.com/charmbracelet/gum#installation) installed first, then:

```sh
curl http://0x0.st/oeMb.sh > ribbot.sh
chmod +x ribbot.sh
./ribbot.sh
```
